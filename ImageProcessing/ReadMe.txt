Pour lancer l'algorithme d'inpainting, il faut lancer main.py. 

En début de fichier, il y a un espace "PARAMETRE UTILISEES", ou vous pouvez modifier l'image utilisée, la taille du patch, et l'écart type du filtre gaussien appliqué avant le calcul du gradient. 

Vous devez ensuite selectionné, la zone que vous voulez supprimer. Une fenêtre s'affichera pour vous montrer la progression de l'algorithme. 

Le code se trouve dans inpainting.py


Des images de tests sont fournit dans le dossier Images
